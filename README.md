## About the Website

This website was created as part of the course project.  
It demonstrates a data analysis exploring the relationships between **BMI, stress, and occupation** on **sleep health** using the *Sleep Health and Lifestyle Dataset* from Kaggle.  
The project includes:
- Data cleaning and wrangling using **R (tidyverse)**
- Visualizations created with **ggplot2**
- Interpretations and summaries formatted in **Quarto**
